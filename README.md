训练模型:train_example.py 可以自定义模型训练的参数.在脚本文件中输入应填的参数就可以在pytorch环境中训练
推理模型:
train_example.py 单个视频的推理.
track_counting_batch.py 批量视频的推理,输入要推理的文件夹后填写要推理的视频数量(按顺序推理自动检测),之后会自动开始推理并输出推理的视频在同一个文件夹内.
视频推理起始线默认视频中线,用户可以自定义起始线的位置.左上角有着简单的计数模块,检测推理框中点经过起始线就会引起计数模块的变化.
predict_example.py 视频预测脚本 — 逐帧检测输出图片，每12帧合并为一张3x4宫格推理图片.可以自定义合并图片的行和列数.
dataset_diagnose.py 利用Laplacian方差(清晰度指标),自动分层不同质量的图片
